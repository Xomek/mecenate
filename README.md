# Mecenate - Тестовое задание

Приложение ленты публикаций для платформы Mecenate (React Native + Expo).

## 🚀 Быстрый старт

### Требования

- Node.js 18+
- npm или yarn
- Expo Go на устройстве (iOS/Android)

### Установка

```bash
git clone <repository-url>
cd mecenate

npm install

npx expo start или npm start
```
